to create a image in docker - docker build -t test-auth-service -f docker/dev/Dockerfile .
to create network between two containers - docker network create [netowk name]
to connect two containers in a same network 
   - EX: mongo (docker run -d --name mongoservice -p 27071:27017 -e MONGO_INITDB_ROOT_USERNAME=harry -e MONGO_INITDB_ROOT_PASSWORD=harry123 --net scout-network mongo)
   - EX: auth-service (docker run -d --name authservicetest -p 5010:5010 -e PORT=5010 -e MONGO_URI=mongodb://harry:harry123@mongoservice --net scout-network test-auth-service)

watch mode working on localhost but need to setup in compose file that will refresh containers
- need to add volumes   

to create mongo container with network - (CMD)
      docker run --rm -it --network db-global -v "%cd%:/usr/src/app" -v "/usr/src/app/node_modules" --env-file "%cd%/.env.dev" -p 5010:5010 -e NODE_ENV=dev auth-service:dev (working)

to create a service image with same db network: (up)
      docker run --rm -it --network app-network -v "%cd%:/usr/src/app" -v "/usr/src/app/node_modules" --env-file "%cd%/.env.dev" -p 5011:5011 -e NODE_ENV=dev auth-service:dev

to create mysql db image: 
      docker run --rm --name mernsql-container -e MYSQL_DATABASE=mern-auth-db -e MYSQL_ROOT_PASSWORD=harry123 -v mernsqldata:/var/lib/mysql -p 3306:3306 -d mysql
